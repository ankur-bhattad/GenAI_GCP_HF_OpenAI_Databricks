Run Streamlit in Google Colab
1. Install dependencies
Run this in a Colab cell:

!pip install streamlit pyngrok
2. Create a Streamlit app file (e.g., app.py)
Create the file inside the Colab VM (not locally):

%%writefile app.py
import streamlit as st

st.title("Hello from Streamlit in Colab")
st.write("This is running inside Google Colab with ngrok tunnel.")
3. Start Streamlit with pyngrok

from pyngrok import ngrok
import subprocess

# Kill any existing streamlit processes
!pkill streamlit

# Start a tunnel on the default Streamlit port 8501
public_url = ngrok.connect(port=8501)
print(f"Streamlit app is live at: {public_url}")

# Run streamlit app
!streamlit run app.py &>/dev/null &
After running this, you'll get an output like:

Streamlit app is live at: http://xyz123.ngrok.io
Click that link to view your Streamlit app live in the browser.

Optional Cleanup (when done)
To stop the app and close tunnels:

!pkill streamlit
ngrok.kill()
